SmartColumnScroller

plugin for jQuery

Description

In multicolumn page layout, automatically scrolls smaller (lower) column so user never sees white space underneath.

Usage


jQuery(window).load(function () {
  jQuery('#myDiv1, #myDiv2').smartColumnScroller();
});

Recommendation

Start plugin processing in 'window.load' event and not in 'document.ready'.

Source code

Source code is available on https://github.com/dhlavaty/jQuery-SmartColumnScroller
